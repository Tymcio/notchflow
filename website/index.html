FAQ

Which Macs does NotchFlow support?

NotchFlow requires a notched MacBook (macOS 14+, Apple Silicon). With an external display attached, the island follows your cursor — on a screen without a hardware notch it appears as a capsule at the top center. It does not run on Macs without a notch (Mac mini, iMac, Mac Studio, older MacBooks without a cutout) — the app will not launch and shows an alert instead.

Does NotchFlow collect my data?

No. Version 1.0 sends no telemetry. Network is used only for update checks and premium license validation.

Can I build the app myself?

Yes. Source code is on GitHub under GPL-3.0. Self-build runs all free features without restrictions.

What does premium include?

Island customization, lyrics, camera mirror, app notifications in the notch, extended shelf, unlimited notes, and more. See full comparison.
