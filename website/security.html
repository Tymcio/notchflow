Security

Official distribution

Download NotchFlow only from notchflow.eu or GitHub Releases published by the notchflow organization.

We never ask you to run Terminal commands, disable Gatekeeper, or remove quarantine attributes.

Verify the app

spctl -a -vv /Applications/NotchFlow.app
codesign -dv --verbose=4 /Applications/NotchFlow.app

Build transparency

Source code is public. You can audit permissions, networking, and licensing logic before building locally.
